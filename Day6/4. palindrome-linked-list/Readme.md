<h1>:heavy_check_mark: 234. Palindrome Linked List</h1>
<blockquote>Given the head of a singly linked list, return true if it is a palindrome.
</blockquote><br>

* **Example 1**:<br>

        Input: head = [1,2,2,1]
        Output: true
      
* **Example 2**:<br>

        Input: head = [1,2]
        Output: false


### **Constraints**

        The number of nodes in the list is in the range [1, 105].
        0 <= Node.val <= 9
